# Calculadora-Brandon
Una calculadora genial


Este proyecto es una calculadora sencilla hecha en Python que permite realizar operaciones de suma, resta, multiplicación y división. Es un proyecto simple y pensado para fines educativos.

## Instalación
1. Instala Python 3.
2. Descarga o clona el repositorio:
   ```
   git clone https://github.com/tuusuario/calculadora-python-basica
   ```
3. Entra al directorio del proyecto:
   ```
   cd calculadora-python-basica
   ```

## Uso
Ejecuta el programa con:
```
python calculadora.py
```
